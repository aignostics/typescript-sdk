      ): Promise<RunReadResponse[]>

      Retrieve a list of application runs with optional filtering

      This method fetches application runs (processing jobs) that have been submitted to the platform. You can optionally filter by application ID and/or application version to narrow down the results.

      Parameters

      • Optionaloptions: { applicationId?: string; applicationVersion?: string }

        Optional filtering parameters

        • OptionalapplicationId?: string

          Filter runs by specific application ID

        • OptionalapplicationVersion?: string

          Filter runs by specific application version

      Returns Promise<RunReadResponse[]>

      A promise that resolves to an array of application run objects

      Throws

      If the request fails due to network issues, authentication problems, or API errors

      Example

      const sdk = new PlatformSDKHttp({ tokenProvider: () => 'your-token' });

      try {
        // Get all runs
        const allRuns = await sdk.listApplicationRuns();

        // Get runs for a specific application
        const appRuns = await sdk.listApplicationRuns({
          applicationId: 'app-123'
        });

        // Get runs for a specific application version
        const versionRuns = await sdk.listApplicationRuns({
          applicationId: 'app-123',
          applicationVersion: 'v1.2.0'
        });

        console.log(`Found ${allRuns.length} total runs`);
      } catch (error) {
        console.error('Failed to list application runs:', error.message);

    cancelApplicationRun

    • cancelApplicationRun(applicationRunId: string): Promise<void>

      Cancel a running or queued application run

      This method attempts to cancel an application run that is currently in progress or waiting in the queue. Once cancelled, the run cannot be resumed and will transition to a cancelled state.

      Parameters

      • applicationRunId: string

        The unique identifier of the application run to cancel

      Returns Promise<void>

      A promise that resolves when the cancellation request is successful

      Throws

      If the request fails due to network issues, authentication problems, invalid run ID, or if the run cannot be cancelled

      Example

      const sdk = new PlatformSDKHttp({ tokenProvider: () => 'your-token' });

      try {
        await sdk.cancelApplicationRun('run-123');
        console.log('Run cancellation requested successfully');

        // Check the status to confirm cancellation
        const run = await sdk.getRun('run-123');
        console.log(`Updated status: ${run.status}`);
      } catch (error) {
        console.error('Failed to cancel run:', error.message);
      }
